# Skoop Clock App

A modern, responsive clock application featuring:
- **Digital Clock**: Local time with date display.
- **World Clocks**: Real-time clocks for London, New York, Tokyo, and Sydney.
- **Stopwatch**: Precision stopwatch with lap functionality.
- **Timer**: Customizable countdown timer with completion alerts.

## Design
- **Theme**: Dark mode with teal (#00b7af) accents.
- **Typography**: Inter for UI and JetBrains Mono for time displays.
- **Performance**: Optimized with cache-control and versioned assets.

## Built With
- Vanilla JavaScript
- CSS3 (Flexbox & Grid)
- HTML5
